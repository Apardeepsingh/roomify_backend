How it works. Storage score is real, derived from each item's height vs. its floor footprint (tall + small base = efficient). Room types are assigned per category. The +/- badge updates per room: storage matters more in an office than a living room, so the same item can flip.
Recommender: filters to the selected room, drops categories you already own, then ranks by what the room needs, with style match as the tie-breaker.
